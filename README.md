# AppManager
iOS device App manager for Windows

## Requirements  
iOS Device (iOS 2 through 15 should work)  
Windows Vista SP2 or newer   
.NET Framework 4.5 or newer  
iTunes 10 or newer (NOT from Microsoft store)
